## PageView Attributes

| 属性  | 说明           | 类型    | 可选值 | 默认值 |
| ----- | -------------- | ------- | ------ | ------ |
| fluid | 是否为流式布局 | boolean | -      | -      |

## PageView Slots

| 插槽名 | 说明             | 参数 |
| ------ | ---------------- | ---- |
| -      | 展示的内容       | -    |
| footer | 自定义底部的内容 | -    |
| popup  | 弹出层放置区域   | -    |
